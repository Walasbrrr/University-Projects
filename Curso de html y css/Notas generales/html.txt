- **La Web no sería posible sin el **HiperTexto**.== **HiperTexto** es lo que permite a los usuarios saltar de una página web a otra a través de enlaces.**

**En esta lección, aprenderás a conectar páginas web con **hipervínculos****

**HiperTexto** es texto que contiene un enlace a otra página.

Las páginas web se llaman documentos **HiperTexto** porque están conectadas por Hipervinculos

**Los hipervínculos** permiten a los usuarios:

1. moverse de una página web a otra

2. compartir información entre diferentes máquinas y sistemas

**La etiqueta de anclaje <a>** se utiliza para crear un hipervínculo en una página web.

**La etiqueta <a>** es una etiqueta contenedora.

eso significa que requiere tanto etiquetas de apertura como de cierre

Los hipervínculos se utilizan para enlazar una página web con otras. Para crear un enlace, necesitas "href" para agregar la URL de destino.

**href** significa referencia **HiperTexto**.

Completa la línea de código para crear un hipervínculo

<a href ="url">Z</a>

La URL debe estar encerrada entre comillas para funcionar sin errores. Si olvidas agregar **href** o las comillas, la etiqueta <a> no creará un hipervínculo.

<a href= 'url'>Z</a>

También puedes utilizar hipervínculos para conectar páginas del mismo sitio web.

La parte de la URL que se mantiene igual para todas las páginas de un sitio web es el **nombre de dominio** (por ejemplo, sololearn.com).

HTML significa **HyperText Markup Language.**

El término "HyperText" se debe a que las páginas web tienen enlaces.

"Markup" se refiere a los símbolos como</>. ////////////////////////////

Los atributos modifican los elementos HTML añadiendo información adicional.

En esta lección, aprenderás a personalizar elementos con sus atributos.

Los **atributos** pueden ser **opcionales** o **requeridos**. Ya conoces algunos atributos requeridos. La etiqueta de imagen requiere un atributo para funcionar correctamente.

Completa la etiqueta de imagen

<img src="http://www.img.png">

Para las etiquetas contenedoras, los atributos siempre van en la etiqueta de apertura. La etiqueta de enlace es otro ejemplo de un elemento HTML que requiere un atributo para funcionar correctamente.

Completa la etiqueta de enlace

<a href="https://z.com">Z</a>

En las etiquetas contenedoras, los atributos siempre van en la etiqueta de apertura

Es una buena práctica describir lo que muestran las imágenes. El atributo **alt** (texto alternativo) se utiliza para agregar descripciones de imágenes.

<img src="img_boy.jpg" 
	 alt ="Boy in a hat">

El atributo **alt** se puede...

- leer en voz alta por lectores de pantalla

- mostrar cuando la imagen no se carga

- leer por motores de búsqueda


Agregar **texto alternativo** a las imágenes hace que tu página sea más accesible

Los atributos generalmente se presentan en pares de nombre/valor como:

```HTML
name="value"
```

Une los ejemplos:

Nombre del atributo: src

Valor del atributo: "http:www.image.png"

Un **sitio web de una sola página** tiene todo su contenido en la página de inicio. Cualquier enlace de navegación lleva al visitante a diferentes secciones (en lugar de cargar nuevas páginas).

Una página de inicio debe llamarse **index.html** para que el navegador web la encuentre y la cargue.

Es una buena práctica nombrar tu página de inicio como **index.html** para que el navegador web pueda encontrarla y cargarla.

RUTAS en HTML

rutas absolutas: son las rutas donde ponemos algo y es absoluto por donde se mire ejemplo, una foto en intagrama, un video de youtube, cosas absolutas (en general lo que esta subido ya en )

Rutas relativas: son las rutas que dependientes de donde estan puedes acceder o no ejemplo: 

si teemos un index.html y queremos acceder a desde un archivo en la misma carpete deberiamos poner:

<a href= "index.html">Nombre de la ruta que se muestra al usuario</a>

pero estas rutas son relativas por el hecho de que puedes moverlas entre carpetas por ejemplo: si mi carpeta de index la pusiera una carpeta antes deberia usar:

<a href="../index.html">Nombre de la ruta que se muestra a el usuario></a>

pero en el caso de que fuera una carpeta adelante seria:

<a href="Nombre de la carpeta donde se encuentra/index.html">Nombre de la ruta que se muestra a el usuario></a>

- The `<!DOCTYPE html>` declaration defines that this document is an HTML5 document
- The `<html>` element is the root element of an HTML page
- The `<head>` element contains meta information about the HTML page
- The `<title>` element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
- The `<body>` element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
- The `<h1>` element defines a large heading
- The `<p>` element defines a paragraph

- - All HTML elements can have **attributes**
- The `href` attribute of `<a>` specifies the URL of the page the link goes to
- The `src` attribute of `<img>` specifies the path to the image to be displayed
- The `width` and `height` attributes of `<img>` provide size information for images
- The `alt` attribute of `<img>` provides an alternate text for an image
- The `style` attribute is used to add styles to an element, such as color, font, size, and more
- The `lang` attribute of the `<html>` tag declares the language of the Web page
- The `title` attribute defines some extra information about an element
